# Clínica Médica Popular Vida+Saúde

Repositório contendo o projeto da **Atividade 4 – Banco de Dados**, incluindo:

- Modelagem do Banco (DER)
- Criação das tabelas (CREATE TABLE)
- Inserção de dados (INSERT)
- Consultas SQL (SELECT)
- Atualizações (UPDATE)
- Exclusões (DELETE)

Todo o projeto foi desenvolvido para o sistema de gestão da **Clínica Médica Popular Vida+Saúde**, contemplando pacientes, médicos, consultas, exames solicitados, resultados de exames, pagamentos e ficha médica.
